# Getting Started

O pacote `@queoponents/tokens` centraliza todos os design tokens do sistema Queoponents: cores, tipografia, espaçamentos, bordas, sombras e animações. Ao consumir os tokens via JavaScript/TypeScript, qualquer mudança feita no Design System é propagada automaticamente para todos os projetos que utilizam a biblioteca.

---

## Instalação

Instale o pacote como dependência do seu projeto:

```bash
# npm
npm install @queoponents/tokens

# yarn
yarn add @queoponents/tokens

# pnpm
pnpm add @queoponents/tokens
```
